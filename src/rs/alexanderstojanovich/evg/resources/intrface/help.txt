Camera movements:
W, UpArrow = forward
A, DownArrow = backward
S = strafeLeft
D = strafeRight
LeftArrow = turnLeft
RightArrow = turnRight

Editor functions:
N = createNewBlock
M = changeBlockColor
Mouse1 = select
Mouse2 = placeNewBlock
1-6 = selectAdjacent
F, 0 = deselect
R = removeSelected
[, ] = changeTexture

Other functions:
` = enterCommand
ESC = mainMenu
F1 = help
F2 = saveLevel
F3 = loadLevel
P = changeCrosshairColor
